Компиляция

Компиляция файла с библиографией происходит с помощью комбинации следующих команд:

pdflatex sixth.tex
bibtex sixth.aux
pdflatex sixth.tex
pdflatex sixth.tex

Я назвал файлы именем sixth.tex и sixth.bib. Команда bibtex sixth.aux ищет файл sixth.bib для оформления библиографии.
